# 6keta-tokei

名前からして地雷くささが漂う処女作です。  
度々の改造を重ね、眩しかったRGBLEDをオフにできるようになっています。

*機能としては*

- 時刻を表示できる
- ボタンを押すと温度を表示する（制度が酷い）
- 時刻をRTCを使って保存しているので電源を落としても時間が正確
- ゲーミングPCからはぎ取ったRGBLEDを無理やり駆動して光る
  - 時間同期失敗時などに色が変わる
- 定期的に時間をインターネット経由で同期する
